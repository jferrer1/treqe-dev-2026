## 🎯 **CRITERIOS DE CALIDAD (NIVEL DOCUMENTO REFERENCIA):**

### 1. NIVEL DE DETALLE
- **NO conciso** - Explayarse, desarrollar ideas completamente
- **SÍ detallado** - Incluir datos, ejemplos, análisis profundos
- **Estructura completa** - Mínimo 5-7 subsecciones por sección principal
- **Longitud adecuada** ~38-42KB por sección (Word con formato)

### 2. CONTENIDO PROFESIONAL
- **Datos reales** 2025-2026 (mercado, usuarios, transacciones)
- **Ejemplos concretos** (Ana, Carlos, Beatriz, David)
- **Análisis competitivo** cuantitativo (benchmarks)
- **Métricas específicas** (CAC, LTV, retención, márgenes)
- **Escenarios múltiples** (pesimista, base, optimista)

### 3. SKILLS A APLICAR (EVALUAR POR SECCIÓN)
Para CADA sección, evaluar y aplicar:
- `humanizer` - Lenguaje natural, profesional, NO robótico
- `legal` - Aspectos legales, protección, estructura
- `business-model-canvas` - Modelo estratégico estructurado  
- `marketing-mode` - Estrategia crecimiento, métricas
- `frontend-design` - Experiencia usuario, diseño
- `algorithm-solver` - Explicaciones técnicas cuando aplique
- Otras skills según temática de la sección

### 4. ESTRUCTURA DOCUMENTO
```
7. EQUIPO Y PLAN DE EJECUCIÓN
  7.1 Equipo Fundador (experiencia, roles, complementariedad)
  7.2 Estructura Organizativa (organigrama, contrataciones plan)

```

### 5. PROCESO DE CREACIÓN (PASO A PASO)
1. **Leer documento de referencia** - Entender nivel de detalle
2. **Evaluar sección** - Qué skills necesita específicamente
3. **Crear estructura** - Mínimo 5-7 subsecciones numeradas
4. **Desarrollar contenido** - Explayarse, no ser conciso
5. **Aplicar skills** - Todas las que correspondan
6. **Verificar contra criterios** - ¿Cumple todos los puntos?
7. **Guardar ÚNICA versión** - Nombre estándar: `0X_NOMBRE_SECCION.docx`

### 6. NOMENCLATURA Y ORGANIZACIÓN
- **Única versión por sección**: `07_EQUIPO_PLAN_EJECUCION.docx`
- **NO crear múltiples versiones**: `_mejorado`, `_concisa`, etc.
- **Eliminar versiones anteriores** si se recrea
- **Mantener consistencia**: Mismo formato, estilo, nivel detalle
