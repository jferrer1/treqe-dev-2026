# Landing Page Treqe - Intercambios Circulares

Landing page moderna y efectiva para captar early adopters para Treqe, la plataforma de intercambios circulares.

## 🎯 Objetivo

Captar emails para waitlist y explicar el valor único de Treqe:
- **Problema:** Intercambios circulares de 3+ personas son imposibles en mercado tradicional
- **Solución:** Treqe los hace posibles con algoritmo de matching circular
- **Beneficio:** 15x más intercambios posibles (75% vs 5% probabilidad)
