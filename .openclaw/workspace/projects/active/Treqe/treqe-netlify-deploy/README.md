## 🚀 Cómo desplegar en Netlify

### Método 1: Arrastrar carpeta (Recomendado)
1. Ve a https://app.netlify.com
2. Arrastra ESTA carpeta completa al área "Drag and drop your project folder"
3. Netlify despliega automáticamente

### Método 2: Subir ZIP
1. Comprime esta carpeta como ZIP
2. En Netlify: "Browse to upload"
3. Selecciona el archivo ZIP

### Método 3: GitHub (Opcional)
1. Crea repo GitHub
2. Sube estos archivos
3. En Netlify: "Import from Git" → GitHub
